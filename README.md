# 🛍️ Myntra Sales Data Analysis & Dashboard

## 📌 Project Overview

This project focuses on analyzing a Myntra dataset to extract meaningful business insights using Microsoft Excel. The workflow includes data cleaning, transformation, analysis using pivot tables, and building an interactive dashboard for visualization.

---

## 🎯 Objectives

* Clean and prepare raw data for analysis
* Identify sales trends and patterns
* Analyze product performance and customer behavior
* Create an interactive dashboard for easy understanding of insights

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Conditional Formatting
* Data Cleaning Techniques

---

## 🔄 Workflow

1. **Data Collection**

   * Imported Myntra dataset

2. **Data Cleaning**

   * Removed duplicates
   * Handled missing values
   * Standardized formats

3. **Data Analysis**

   * Created pivot tables
   * Identified key metrics and trends

4. **Data Visualization**

   * Applied conditional formatting
   * Designed charts and graphs

5. **Dashboard Creation**

   * Built an interactive dashboard for insights

---

## 🔍 Key Insights

* Identified top-performing products/categories
* Observed sales trends over time
* Highlighted high and low-performing segments
* Provided actionable insights for better decision-making
